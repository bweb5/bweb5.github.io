# ECE 595 – Autonomous Mobile Robots  
### Benjamin Webster & Kevin Tran  

This repository contains the official class website for our Spring 2026 ECE 595 Autonomous Mobile Robots project.

The purpose of this repository is strictly to host project documentation, weekly progress updates, and high-level system descriptions for our vision-based UAV tracking system.

---

## About This Repository

This repository is **exclusively** used for:

- Hosting the GitHub Pages website
- Publishing weekly project updates
- Documenting system architecture and development progress
- Providing a public overview of the project

This repository does **not** contain the primary project source code.
